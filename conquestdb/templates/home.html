ConquestDB.com

A deckbuilder and database website for the Warhammer 40,000: Conquest LCG.

Welcome to conquestdb.com!

This site serves as a deckbuilder for the Warhammer 40,000: Conquest Living Card Game.

For information about how to play the game, please see the links below.

FFG Learn to Play

FFG Rules Reference

If you would rather watch a video, you can find a tutorial by The Hive Tyrant here.

All additional rules, such as those found in expansions or introduced by Apoka Team, can be found at:

Apoka and Expansions rules collection

For playing the game online, there is a Tabletop Simulator (TTS) mod, and a website found respectively at:

TTS Mod

iridial.net

This website supports exporting decks to both the TTS mod and iridial.net.

The source code for this website can be found on the GitHub.
